The Sign Language Animation Tool is a project designed to convert text or input into sign language animations.
It aims to improve accessibility and communication for people with hearing or speech impairments.

##🚀 Features:
1.Convert text into sign language animations
2.Speech-to-text input using microphone
3.Word-by-word animation playback
4.Extracts key words from sentences
5.User authentication (Sign Up / Login)

##🛠️ Technologies Used
- Python
- Django (Web Framework)
- HTML5
- CSS3
- JavaScript
- Web Speech API (Speech Recognition)
- Git & GitHub

  ##⚙️ Working
- User enters text or uses microphone input
- The system extracts key words from the sentence
- Each word is mapped to a corresponding sign language video
- Videos are played sequentially to form an animation
